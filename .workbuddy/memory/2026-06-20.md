## Bug 修复记录 (6/21 凌晨 1:50)

### 首页全部显示"加载中"
- **根因**: dashboard.js 的 renderCatStatus 函数中，单引号字符串内包含 `&` 字符导致 JavaScript 语法解析错误（SyntaxError: Unexpected token ';'），整个 dashboard.js 文件无法被浏览器加载
- **修复**: 将该函数内所有字符串改为双引号包裹，中文使用 Unicode 转义，JS 版本号升级到 v=3
- **教训**: JS 字符串拼接中避免在单引号内使用 & 等特殊字符，统一用双引号或 Unicode 转义
