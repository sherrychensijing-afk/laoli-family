# 2026-06-20 工作记录

## 老郦家 家庭管理系统 - 初版完成

### 技术栈
- 前端: HTML + CSS + Vanilla JS (零框架)
- 后端: Node.js 22.12.0 + Express.js + JSON 文件持久化
- 端口: 3456

### 已实现功能
1. **首页概览**: 家庭成员卡片、今日待办、猫咪状态
2. **日程提醒**: 自动计算引擎支持:
   - 郦赟每周五/日航班 (含节假日调整)
   - 提前30天购票提醒
   - 猫咪每2天冻干喂食 + 每3天水粮检查
   - 每日亲亲提醒
   - 手动添加/完成/删除日程
   - 分类筛选 (出行/猫咪/亲密/自定义)
3. **郦家家规**: 预置8条家规，支持在线编辑/新增/删除，实时持久化

### 数据文件
- `/data/family.json` - 家庭信息
- `/data/schedules.json` - 日程+完成记录
- `/data/rules.json` - 家规
- `/data/feed-log.json` - 喂食日志

### 启动方式
```bash
cd /Users/chensijing/WorkBuddy/2026-06-20-23-15-15
NODE_PATH=... node server.js
```
访问 http://localhost:3456
